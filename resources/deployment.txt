1. Create a fat jar with jackson - mvn clean package

2. Create a custom image with bundled JRE and JavaFX - jlink --module-path "%JAVA_HOME%\jmods;C:\Program Files\Java\javafx-sdk-21\jmods" --add-modules java.base,javafx.controls,javafx.media --output target/DeadShock

3. Create executable exe - jpackage --type app-image --name DeadShock --input target --main-jar DeadShock-1.0.0.jar --main-class com.abhai.deadshock.Game --runtime-image target/DeadShock --icon resources/images/icon.ico
